Features

Everything you need for the perfect chart

No MIDI knowledge. No manual charting. Just a Spotify link and one command.

🎧

Spotify → Chart

Paste any Spotify track URL and SongHero finds the audio, downloads it, and generates a full chart. No API keys, no setup — just the link.

🤖

Gemini AI Engine

Google Gemini 2.0 Flash maps song sections — verses, choruses, bridges — and builds note patterns that feel like a human charted every phrase.

🎚️

4 Difficulty Levels

Easy, Medium, Hard, and Expert — each with independent note grids. Progressive orange fret and chord usage tuned to the song's actual intensity.

📺

Music Video Included

Automatically downloads the official music video and packages it as a video background — just like Clone Hero expects. Drop in and play.

🥁

Beat-Accurate Timing

librosa audio analysis extracts precise onsets, tempo maps, and pitch curves. Every note lands on an actual beat — no random placement.

📦

Ready-to-Play Output

Outputs a complete Clone Hero song folder with notes.chart, song.ini, album art, and audio. Move it to your Songs folder and you're done.

Pipeline

Four steps. Zero effort.

From a Spotify URL to a playable chart folder in under a minute.

01

Resolve

Extracts song title, artist, album art, and metadata from the Spotify URL using public info — no auth required.

02

Download

Fetches the highest-quality audio and optional music video from YouTube via yt-dlp and ffmpeg.

03

Analyze

librosa detects beats, onsets, and pitch. Gemini maps song sections and suggests note patterns per difficulty.

04

Generate

Builds a .chart file with all 4 difficulties, writes metadata, and packages everything into a ready-to-play folder.

Install

One command. That's it.

Works on macOS and Linux. Windows via WSL.

$ curl -sSL https://raw.githubusercontent.com/jackwallner/portfolio/main/docs/songhero/install.sh | bash
$ git clone https://github.com/jackwallner/any-song-clone-hero-cli.git
$ cd any-song-clone-hero-cli && chmod +x index.js
Prerequisites: brew install yt-dlp ffmpeg · pip3 install librosa soundfile numpy scipy
