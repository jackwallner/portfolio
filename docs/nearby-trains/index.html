Nearby Trains

Real-time tracker for Amtrak, VIA Rail, and Brightline trains near any location. No build step, no framework, no API key.

Open the live demo ↗ View source ↗

§ The pitch

Built in memory of my grandfather, who loved trains. He'd want to know which one was passing closest to him right now.

Set your location — GPS, city search, or coordinates — and the page polls the free Amtraker v3 feed every sixty seconds, computes proximity and closest-approach for every active passenger train in North America, and logs every train that passes within your detection radius. Every spot persists in localStorage; export to JSON whenever.

§ Why no framework

This site has seven JS modules — storage, amtraker, location, tracker, map, ui, app — totalling a few hundred lines. Importing React would multiply that footprint by twenty without doing any work the browser doesn't already do natively.

Modern ES modules, the Fetch API, localStorage, and a 40 KB map library cover everything. Page weight is small enough to load on a phone with one bar.

§ Why no freight trains

The honest answer is: there's no public feed. Unlike aviation, where ADS-B is broadcast in the clear and anyone with a $20 dongle can decode it, U.S. freight railroads (BNSF, UP, CSX, NS, CN, CP) run their Positive Train Control on encrypted private spectrum and publish zero public position data. There's nothing to scrape and nothing to subscribe to. So the app shows passenger trains, which do have an open feed via Amtraker, and notes the limitation honestly in the README.

A small site, built carefully, that does one thing and links out to its sources.
