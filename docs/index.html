No. 03

Streak FinderIn TestFlight

2026 · iOS, watchOS

Mines four-hundred days of your Apple Health history to surface the streaks you didn't realize you had — across nine metrics, every threshold tier, daily and weekly. Picks the highest-tier streak per metric, ranks them by an interestingness score, and protects the hero one with a 7 PM at-risk nudge. A weekly streak holds the line when the daily one breaks.

No. 04

Kalshi BTC BotPrivate

2026 · Python, GH Actions

An unattended trading bot for Kalshi's hourly Bitcoin "price-above" markets. Authenticates with an RSA-signed API request, pulls the active hour's market, takes a sentiment read off the Fear & Greed Index, and either buys YES, buys NO, or sits out — capped at one dollar of risk per fifteen-minute run. Runs on a GitHub Actions cron and writes every decision to a structured trade log.

Python · RSA-PSS auth · REST · GitHub Actions cron · structured logs

No. 06

Nearby TrainsLive demo

2026 · Web, no-build

Real-time tracker for Amtrak, VIA Rail, and Brightline trains near any location. No build step, no framework — just vanilla ES modules, Leaflet, and the free Amtraker v3 API.
